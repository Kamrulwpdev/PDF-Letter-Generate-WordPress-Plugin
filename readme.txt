== Description ==
WP Letter Generator WordPress plugin provide an AI Letter Generating System. User can Generate with Input Fill Letter Form to Generate Letter and Download it generted pdf file. This plugin very smooth and use-friendly and easy to use. Include Admin Plugin Setting option to customize color, font's and etc.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Just install and active it will work automaticall.
4. Notice: Go to the plugin Setting Page, there have All Shortcode List.
